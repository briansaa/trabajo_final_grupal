# Institución Educativa Alfred Nobel - Plataforma Web 🎓✨

Un proyecto web interactivo y moderno diseñado para la **Institución Educativa Alfred Nobel**. Esta plataforma proporciona una experiencia de usuario (UX) inmersiva y una interfaz de usuario (UI) de alta calidad, orientada a estudiantes, padres y personal administrativo.

![Versión](https://img.shields.io/badge/Versi%C3%B3n-2.0%20(Modern%20UI)-blue)

## 🌟 Características Principales

El proyecto ha sido recientemente rediseñado para abandonar frameworks antiguos y adoptar un enfoque centrado en **CSS moderno (Flexbox, Grid)** y **Glassmorphism** (Efecto de cristal esmerilado).

*   **Página de Inicio (`index.html`):** Tarjetas interactivas con efectos *hover* y una presentación impactante de los niveles educativos (Inicial, Primaria, Secundaria).
*   **Información Institucional (`institucion.html`):** Diseño de contenido con barra lateral adhesiva (*sticky sidebar*) para navegar fácilmente por la misión, visión, historia y el mapa de ubicación interactivo.
*   **Secretaría y Cálculo de Notas (`registro de notas2.html`):** Una interfaz tipo "dashboard" que incluye:
    *   Una **calculadora widget** estilizada estilo iOS.
    *   Un sistema interactivo para ingresar promedios de alumnos.
    *   Generación de reportes de notas dinámicos en tiempo real con JavaScript nativo.
*   **Centro de Contacto (`contactenos.html`):** Formulario estético translúcido (*Glassmorphism*) enfocado en la usabilidad y la retroalimentación visual al usuario.
*   **Galería Multimedia (`galeria.html`):** Cuadrícula de fotos avanzada en *CSS Grid* (estilo mosaico/Pinterest) con un visualizador de imágenes (Lightbox) incorporado sin dependencias externas.

## 🛠️ Tecnologías Empleadas

Este proyecto está construido íntegramente con tecnologías fundamentales de la web de forma nativa:

*   **HTML5 Semántico**: Para una estructura sólida y accesible.
*   **CSS3 Moderno**: Empleando variables nativas, `CSS Grid`, `Flexbox`, transformaciones, transiciones y el efecto `backdrop-filter` (Glassmorphism).
*   **JavaScript (Vanilla JS)**: Para el simulador de cálculo de notas, manipulación del DOM interactivo en las galerías, validaciones de formularios y el comportamiento del menú móvil.
*   **SweetAlert**: Para gestionar alertas estilizadas amigables (*pop-ups*) al completar cálculos o formularios.
*   **Font Awesome 6**: Para la iconografía del sitio web.
*   **Google Fonts**: Tipografías modernas *Poppins* y *JetBrains Mono*.

## 🚀 Cómo Ejecutar el Proyecto Localmente

La mejor parte de este proyecto es que **no requiere instalación de dependencias, compilación, ni servidores pesados (Node.js, etc.)**.

1.  **Clona este repositorio:**
    ```bash
    git clone https://github.com/TU-USUARIO/nombre-de-tu-repositorio.git
    ```
2.  Navega a la carpeta del proyecto.
3.  Simplemente abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Safari, Edge).
